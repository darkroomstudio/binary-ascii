# binascii

> Binary <-> ASCII

```ts
import binascii from '@dkrmstudio/binascii'

console.log(binascii('dkrm')) // '01100100011010110111001001101101'
console.log(binascii('01100100011010110111001001101101')) // 'dkrm'
```
